# Agents

Prompty, AI agenti a jejich registr.

## Struktura

```
Agents/
├── registry.md       → přehled všech agentů
├── prompts/          → systémové prompty jednotlivých agentů
└── workflows/        → vícekrokové AI workflow
```

## Jak přidat nového agenta

1. Vytvoř soubor v `prompts/nazev-agenta.md`
2. Přidej záznam do `registry.md`
3. Otestuj prompt a zapiš výsledky do záznamu v registru
