# Projects

Každý aktivní projekt má vlastní podsložku s kontextem, poznámkami a stavem.

## Struktura projektu

```
Projects/
└── nazev-projektu/
    ├── brief.md          → co, proč, pro koho
    ├── status.md         → aktuální stav, další kroky
    ├── notes.md          → pracovní poznámky, nápady
    └── assets/           → soubory, obrázky, exporty
```
