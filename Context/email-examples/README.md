# Ukázky e-mailů

Skutečné nebo upravené e-maily, které slouží jako reference pro AI.
AI se z nich naučí tvůj styl a může psát e-maily, které znějí jako ty.

## Jak přidat ukázku

1. Zkopíruj e-mail do nového souboru: `situace-popis.md`
2. Anonymizuj citlivá jména/data
3. Přidej záznam do tabulky níže
