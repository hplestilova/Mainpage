# Tools

Skripty, API napojení a opakovaně použitelné nástroje.

## Struktura

```
Tools/
├── scripts/          → automatizační skripty (Python, Bash, JS)
├── api-connections/  → konfigurace a příklady napojení na API
└── templates/        → opakovaně použitelné šablony dokumentů
```

## Přehled nástrojů

| Nástroj | Typ | Popis | Soubor |
|---------|-----|-------|--------|
| _(zatím žádný)_ | — | — | — |

## Konvence

- Skripty mají na začátku komentář s popisem a použitím
- API klíče **nikdy neukládej do tohoto repozitáře** — používej `.env` soubory nebo správce hesel
- Každý skript otestuj a zaznamenej výsledky v jeho hlavičce
